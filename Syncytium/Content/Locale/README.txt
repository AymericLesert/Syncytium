In this directory, you can find out languages used by Webix and moment.

For WEBIX :
-----------

The file can be downloaded from :
http://cdn.webix.com/site/i18n/fr.js

After downloading these files, change the locales by the language code used by the application :
webix.i18n.locales["fr-FR"] > webix.i18n.locales["FR"]

For MOMENT :
------------

The file moment.js contains some features attached to moment about the globalization and some format dates
